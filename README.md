🌸 PuffyWalls — The Cute, Aesthetic Wallpaper Hub
PuffyWalls is a minimalist, fast, and responsive full-stack web application designed for browsing, filtering, and saving cute, high-quality aesthetic wallpapers. Built as a unified modern platform, it leverages the Pexels API for dynamic image discovery alongside a robust custom backend to manage user profiles and save personal favorites.

✨ Features
🌸 Pastel & Cute Aesthetic: A soft, modern user interface utilizing smooth glassmorphic elements and micro-interactions.

⚡ Next.js Architecture: Built using Next.js for efficient page routing, optimized server-side rendering, and fast API integration.

🔄 Infinite Scroll Discovery: Seamlessly loads subsequent rows of images automatically as the user scrolls down the Explore, Home, or Trending grids.

🔍 Live Filtered Search: Instantly query wallpapers by text, tags, and predefined categories with real-time updates.

📱 Live Mockup Previews: Open a detailed modal to see exactly how a wallpaper looks applied directly behind an interactive phone or desktop chassis before downloading.

🔒 Persistent Authentication: Secure user authentication powered by Firebase Auth, utilizing browserLocalPersistence to keep users safely signed in across server restarts and browser refreshes.

🛠️ Tech Stack
Frontend & Styling
Framework: Next.js (App Router)

Styling & Presentation: CSS Grid Layout

Icons & Micro-interactions: Lucide React + CSS Transitions

Sonner: Next.js library for toast notifications

Backend & Authentication
Database Wrapper: Mongoose (MongoDB Object Modeling)

Database Engine: MongoDB (Document-based cloud cluster)

Authentication: Firebase Auth (Client-side engine with persistent tokens)

API Architecture: Next.js Native API Routes

Primary Data Sources & Hosting
Image Engine: Pexels API (On-demand data sourcing)

Deployment Platform: Vercel
